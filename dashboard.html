Help and Troubleshooting

Open the extension popup and use Check Current Page. This sends the active tab URL to the local API and ML services and returns a live confidence-based verdict.
The dashboard preserves your last successful telemetry snapshot, marks status as degraded, and resumes live updates once services become healthy again.
Run Rust API on port 8080, Python ML service on port 8000, and Redis on port 6379 for full threat scoring, queueing, and analytics ingestion.
Open the extension popup once to refresh local control-plane credentials. The dashboard automatically retries token refresh on 401 responses.
High-confidence risky URLs can be auto-blocked when enabled in settings. Live alert mode highlights recent threats with confidence and action data.
Use the time range selector to compare trend velocity, block ratio, and threat mix. Focus on spikes in high-risk classes and repeated domains.
